# 步骤 2 —您的数据需要什么格式？

因此，您首先需要了解的是，您的数据需要是什么格式，以便轻松地使用这个包中的 **sunburst()** 函数。

让我们看一看。你会在例子**中看到。csv 下面有两列:**一列是旅程中每页的**，用连字符“-”**分隔，第二列是****导航通过此旅程的会话总数**。**
