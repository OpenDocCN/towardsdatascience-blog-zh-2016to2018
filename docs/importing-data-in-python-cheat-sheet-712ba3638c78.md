有了这个 Python 备忘单，您将有一个方便的参考指南来导入您的数据，从平面文件到其他软件和关系数据库的本地文件。

本快速指南帮助您学习在 Python 中导入数据的基础知识，您将需要开始清理和整理您的数据！
